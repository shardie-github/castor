## Quick Start Guide

### Step 1: Create Account
1. Visit [app.castor.com/signup](https://app.castor.com/signup)
2. Enter email and password
3. Verify email address
4. Complete onboarding wizard

**Common Issues:**
- **Email not received:** Check spam folder, resend verification
- **Password requirements:** Minimum 8 characters, include number and special character

### Step 2: Connect Podcast
1. Go to Settings → Integrations
2. Select hosting platform or enter RSS feed
3. Authorize connection
4. Wait for episodes to sync (usually <5 minutes)

**Common Issues:**
- **RSS feed invalid:** Verify feed URL, check feed format
- **Episodes not syncing:** Check feed accessibility, verify feed URL
- **Integration not working:** Reconnect integration, check API credentials

### Step 3: Create First Campaign
1. Go to Campaigns → New Campaign
2. Enter campaign details
3. Select episodes
4. Add tracking links
5. Save campaign

**Common Issues:**
- **Campaign not saving:** Check required fields, verify dates
- **Episodes not appearing:** Ensure podcast is connected, check episode dates

### Step 4: Generate Report
1. Go to Campaigns → Select Campaign
2. Click "Generate Report"
3. Customize report (optional)
4. Download PDF

**Common Issues:**
- **Report generation failed:** Check campaign data, verify attribution events
- **Report missing data:** Ensure tracking links are set up, check date ranges

## Common Issues & Solutions

### Podcast Connection Issues

**Problem:** RSS feed not validating
**Solution:**
1. Verify feed URL is correct
2. Check feed is publicly accessible
3. Validate feed format (use feed validator)
4. Ensure feed contains episodes

**Problem:** Episodes not syncing
**Solution:**
1. Check feed last updated date
2. Verify feed accessibility
3. Reconnect integration
4. Contact support if issue persists

### Campaign Issues

**Problem:** Campaign not tracking correctly
**Solution:**
1. Verify tracking links are correct
2. Check attribution model settings
3. Ensure links are being used
4. Review attribution events

**Problem:** ROI calculation incorrect
**Solution:**
1. Verify campaign cost entered correctly
2. Check attribution events are recorded
3. Review conversion values
4. Ensure date ranges are correct

### Report Issues

**Problem:** Report generation failed
**Solution:**
1. Check campaign has data
2. Verify attribution events exist
3. Ensure date ranges are valid
4. Try generating again

**Problem:** Report missing data
**Solution:**
1. Check date ranges
2. Verify tracking links are set up
3. Review attribution events
4. Ensure campaign is active

### Integration Issues

**Problem:** Hosting platform not connecting
**Solution:**
1. Verify API credentials
2. Check platform API status
3. Reconnect integration
4. Contact platform support if needed

**Problem:** Data not syncing
**Solution:**
1. Check integration status
2. Verify API permissions
3. Manually trigger sync
4. Contact support if issue persists

## Error Messages Reference

### Authentication Errors

**Error:** "Invalid email or password"
- Verify email and password are correct
- Check for typos
- Reset password if needed

**Error:** "Email not verified"
- Check email inbox for verification link
- Resend verification email
- Contact support if link expired

### API Errors

**Error:** "API rate limit exceeded"
- Wait 1 minute and retry
- Upgrade to higher tier for higher limits
- Contact support for limit increase

**Error:** "Invalid API key"
- Verify API key is correct
- Regenerate API key if needed
- Check API key permissions

### Data Errors

**Error:** "Campaign not found"
- Verify campaign ID is correct
- Check campaign exists
- Ensure user has access

**Error:** "Invalid date range"
- Verify start date is before end date
- Check dates are valid
- Ensure dates are within allowed range

## Support Processes

### Ticket Creation Process

**Step 1: Search Knowledge Base**
- Check if issue is documented
- Review troubleshooting guides
- Watch video tutorials

**Step 2: Create Support Ticket**
- Go to Support → Create Ticket
- Select category
- Describe issue in detail
- Attach screenshots/logs

**Step 3: Ticket Response**
- Tier 1: <48 hours
- Tier 2: <4 hours
- Tier 3: <2 hours
- Tier 4: <1 hour

- Technical troubleshooting
- Integration problems

**Level 3: Engineering**

- Critical issues

**Level 4: Account Manager (Enterprise)**
