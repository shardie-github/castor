**For:** Disciplined Entrepreneurship Lens, Pricing Strategy  
**Last Updated:** 2024

## Overview

This document documents pricing logic, including assumptions, willingness-to-pay (WTP) data, value metrics, and conversion triggers.

---

## Pricing Strategy: Value-Based Pricing

**Approach:** Price based on value delivered (time saved, revenue increase)

**Rationale:**
- Time savings: 2+ hours → <30 seconds = $116-290/month value
- Revenue increase: 20-40% higher renewal rates = $200-2000/month value
- Professional image: Improved sponsor relationships = intangible value

---

## Pricing Tiers

### Tier 1: Free

**Price:** $0/month

**Target Persona:** Solo Podcaster (early stage), Trial Users

**Features:**
- 1 podcast
- Basic analytics (downloads, streams)
- 1 campaign per month
- Basic report generation (limited templates)
- Community support
- Attribution tracking (promo codes only)

**Limitations:**
- No ROI calculations
- No automated reports
- No API access
- No white-labeling
- Limited historical data (30 days)

**Value Delivered:**
- Basic analytics: $0 value (limited)
- Basic reports: $0 value (manual)

**WTP Data:**
- Solo Podcaster: $0-29/month
- Average: $15/month

**Conversion Triggers:**
- User generates 3+ reports
- User creates 2+ campaigns
- User views dashboard 10+ times
- User exports data 5+ times
- Campaign renewal rate >50%

## Pricing Assumptions

### Value Assumptions
- **Time Value:** $29/hour (average podcaster hourly rate)
- **Time Saved:** 2-10+ hours/week (depending on tier)
- **Revenue Increase:** 20-40% higher renewal rates (depending on tier)
- **Campaign Value:** $500-5000/month (depending on podcaster size)

### Conversion Assumptions
- **Free → Starter:** 10% conversion rate
- **Starter → Professional:** 25% conversion rate
- **Professional → Enterprise:** 15% conversion rate

### Churn Assumptions
- **Monthly Churn:** <5%
- **Annual Churn:** <40%

## Conversion Logic

### Freemium to Paid Conversion

**Trigger Events:**
1. Value Realization: User generates first report
2. Engagement: User views dashboard 5+ times
3. Feature Limitation: User hits free tier limit
4. Time-Based: User active for 7+ days

**Conversion Score:**
- Reports generated >= 3: +20 points
- Campaigns created >= 2: +20 points
- Dashboard views >= 10: +15 points
- Data exports >= 5: +15 points
- Days active >= 7: +10 points
- Renewal rate > 50%: +20 points
- **Threshold:** 50+ points triggers upsell
