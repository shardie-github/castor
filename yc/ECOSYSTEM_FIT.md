**Key AI Components:**

**AI Differentiation:**

---

### 2. B2B SaaS / Enterprise Software
**Fit Level:** High

**Why:**
- Multi-tenant architecture (`src/tenants/`)
- Enterprise features (RBAC, SSO, white-labeling)
- API-first design (`src/api/`)
- Scalable infrastructure (PostgreSQL, TimescaleDB, Redis)

**Enterprise Features:**
- Multi-tenant isolation
- Role-based access control (RBAC)
- Attribute-based access control (ABAC)
- Audit logs
- API access
- White-labeling
- SSO/SAML authentication

**Target Customers:**
- Solo podcasters (SMB)
- Producers/agencies (Mid-market)
- Podcast networks (Enterprise)

---

### 3. Media / Content Technology
**Fit Level:** Medium

**Why:**
- Podcast-specific features (RSS ingestion, episode management)
- Content analysis and analytics
- Attribution tracking for media campaigns

**Media Components:**
- RSS feed ingestion (`src/ingestion/`)
- Episode management
- Listener analytics
- Campaign attribution

**Media Differentiation:**
- Not just analytics, but attribution and matching
- Cross-platform tracking (podcast → website → purchase)
- Media-specific ROI calculations
